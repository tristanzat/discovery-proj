# Realtime Dungeon Crawler

Project led by Tristan Zatylny. Developed with the help of AI.

## Running this

1. Clone this repository
2. Open two terminals, one in `frontend`, one in `backend\DungeonCrawler.API`
3. In the frontend terminal, run `npm install`
4. When that finishes, in the same terminal, run `npm run dev`
5. In the backend API terminal, run `dotnet run --launch-profile http`
6. Open the localhost thing that opens up

That's it
